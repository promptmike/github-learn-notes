# Codespaces versus GitHub.dev editor

You may wonder when you should use Codespaces vs GitHub.dev
Codespaces and GitHub.dev are not the same thing

You can open a repository, fork or pull request from the GitHub.dev editor to make and commit changes
GitHub.dev is an editor for GitHub

If you want to build, test and a run your code, you cannot use GitHub.dev as it has no compute, while Codespaces does have compute
GitHub.dev is an editor for GitHub, while Codespaces is a compute environment that can both edit and run code from GitHub

## Comparison of Codespaces and GitHub.dev

|                 | GitHub.dev                       | GitHub Codespaces                                                                     |
| --------------- | -------------------------------- | ------------------------------------------------------------------------------------- |
| Cost            | Free                             | Monthly quota                                                                         |
| Availability    | All users                        | All users                                                                             |
| Startup         | Opens instantly with a key press | Takes a few minutes to assign a VM and configure the container from devcontainer.json |
| Compute         | None                             | Dedicated VM                                                                          |
| Terminal Access | None                             | Yes                                                                                   |
| Extensions      | Only ones that run on the web    | Most VS Code extensions will work                                                     |

GitHub.dev is an editor available free of charge to all users, while Codespaces is a virtual environment that can install extensions and build and run code within a monthly quota

## Continue working on Codespaces

You can start work on GitHub.dev and continue it on Codespaces
GitHub.dev is a code editor for GitHub with no compute, while Codespaces is an extensible code environment for building, testing and running code

Commit your changes, select *Continue working on...* > *Create New Codespace* to continue your work from GiHub.dev on a Codespace
GitHub.dev is a code editor that runs on the web, while Codespaces is a virtual environment with compute resources
