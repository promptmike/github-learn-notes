## Subscription plans

GitHub Copilot is available on different plans to suit your needs
> Copilot brings Codex-powered autocomplete to GitHub, IDE and CLI on multiple plans

### GitHub Copilot Free

GitHub Copilot Free can be accessed by signing into your GitHub account in Visual Studio Code
> Copilot brings autocomplete to GitHub, IDE and CLI, with a free tier accesible from Visual Studio Code

GitHub Copilot Free allows 2000 code completions and 50 chat requests per month in supported editors
> Copilot brings code completion and chat requests to GitHub, IDE and CLI, with a free plan available

### GitHub Copilot Pro

Pro tier offers unlimited code completions and chat requests, priority access to latest models and automated test generation
> Copilot brings code completion and chat requests to the IDE, with Free and Pro plans for individuals

### GitHub Copilot Pro+

Pro+ offers everything in Pro, plus additional premium model usage and priority infrastructure access
> Copilot brings code completion and chat requests to the IDE, with additional Pro and Pro+ plans for high-usage individuals

### GitHub Copilot Business
